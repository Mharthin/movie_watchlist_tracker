# 🎬 My Watchlist - Movie Tracker

A modern, sleek movie watchlist application built with vanilla HTML, CSS, and JavaScript. Track movies across three statuses, rate them with stars, and organize your viewing journey.

## ✨ Features

- **Add Movies** — Enter movie titles with status and star ratings
- **Three Status Categories**:
  - Want to Watch (Grey)
  - Watching (Yellow)
  - Watched (Green)
- **Star Ratings** — 5-star rating system with visual feedback
- **Live Counters** — Status breakdown and total movie count in header
- **Filter & Search** — View all movies or filter by status
- **Persistent Storage** — localStorage saves your watchlist automatically
- **Delete Movies** — Remove unwanted entries with one click
- **Responsive Design** — Works seamlessly on desktop and mobile
- **Modern UI** — Glass morphism, animations, gradient effects, and smooth transitions

## 🎨 Design Highlights

- **Dark Theme** — Deep black background (#0a0a0a) with off-white text
- **Red Accents** — Netflix-inspired red (#e50914) for interactive elements

- **Smooth Animations** — Fade-in, slide-up, and floating elements

## 🚀 Getting Started

### Option 1: Open Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Mharthin/movie_watchlist_tracker.git
   cd movie_watchlist_tracker
   ```
2. Open `index.html` in your browser
3. Start adding movies!

### Option 2: Live Preview
No setup needed! Just open the HTML file directly in any modern browser.

## 📖 How to Use

1. **Add a Movie**:
   - Enter the movie title
   - Select a status (Want to Watch, Watching, or Watched)
   - Click the stars to rate (1-5)
   - Click "Add Movie" or press Enter

2. **Filter Movies**:
   - Use the filter buttons to view movies by status
   - Click "All" to see everything

3. **Delete a Movie**:
   - Click the ✕ button on any movie card

4. **Track Progress**:
   - Check the header for live status counts
   - See total movie count in the red badge

## 💾 Data Storage

Your watchlist is automatically saved to browser localStorage. It persists even after closing the browser!
